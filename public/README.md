# Public Assets

Place your images and static files here. They will be served from the root path in production.

## Usage Examples:
- `public/images/driver.jpg` → accessed as `/images/driver.jpg`
- `public/logo.png` → accessed as `/logo.png`

## Supported Formats:
- Images: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.svg`
- Videos: `.mp4`, `.webm`
- Documents: `.pdf`, `.txt`
